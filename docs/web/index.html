Little sense is designed specifically for researchers and enthusiasts who want to develope bespoke sensor network applications. The code in this project is intended as a foundation on which specialise applications can be build. Out of the box, Little sense provides a local server (Raspberry Pi based), which can will collect readings collected by Arduino based sensor (example code provided). Sensors can communicate with little sense over Wifi and via other radios such as RFM69. The server also provides a web interface where users can inspect and see visualisations of the collected data.
